## What Was Built

### D1 — `--share` on `agentkit analyze`

- Added `--share` flag to `agentkit_cli/commands/analyze_cmd.py` and `agentkit_cli/main.py`
- When set, generates a scorecard HTML from the analysis result and uploads via `upload_scorecard()`
- URL printed prominently: `"Score card published: https://..."`
- `--json` output includes `"share_url"` field when `--share` is set
- Upload failure is non-fatal: warns to stderr, analysis exits 0

### D2 — `--share` on `agentkit sweep`

- Added `--share` flag to `agentkit_cli/commands/sweep_cmd.py` and `agentkit_cli/main.py`
- Generates a combined scorecard for all analyzed targets using `generate_sweep_scorecard_html()`
- Single upload (not per-target), single URL printed
- `--json` output includes `"share_url"` in summary object
- Upload failure is non-fatal

### D3 — Scorecard HTML improvements for external repos

- Extended `generate_scorecard_html()` in `agentkit_cli/share.py` with optional `repo_url` and `repo_name` params
- Repo name rendered as clickable link in card header when `repo_url` is set
- Analysis timestamp now shown in footer
- "Generated by" changed to "Analyzed by" with PyPI link
- Fully backward-compatible: all existing callers unaffected (optional params)
- Added `generate_sweep_scorecard_html()` for multi-target combined scorecards

### D4 — README + docs + version bump

- README: added `--share` examples for `analyze` and `sweep` in Sharing Results section
- CHANGELOG: v0.25.0 section with D1-D3 summary
- Version bumped: `agentkit_cli/__init__.py` and `pyproject.toml` → 0.25.0
- Updated version assertions in `tests/test_main.py` and `tests/test_leaderboard.py`
