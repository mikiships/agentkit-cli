The Full Toolkit

coderace

agentmd

Auto-generate CLAUDE.md / AGENTS.md. GENERATE context files.

agentlint

Static analysis for agent context files. GUARD freshness.

agentreflect

agentkit-mcp

agentkit-cli

One CLI to orchestrate the full pipeline.

Key Commands

CommandDescription
agentkit runRun full quality pipeline on your repo
agentkit analyze github:owner/repoScore any public GitHub repo instantly
agentkit quickstartZero-config first-run experience
agentkit daily --pagesUpdate the daily leaderboard on GitHub Pages
agentkit trending --pages-trendingPublish trending repos scorecard
agentkit org github:ownerScore all public repos in a GitHub org
agentkit user-scorecard github:userDeveloper Profile Card: grade any developer's repos

Daily Trending

See which trending GitHub repos are most AI-agent-ready today.

Run agentkit trending --pages-trending to update. Subscribe to stay current.

View trending leaderboard →

Org Leaderboard

Rank all public repos in any GitHub org by agent-readiness score.

Run agentkit org github:owner --pages-org to publish your org's leaderboard.

View org leaderboards →

Developer Profile Card

Grade any developer's public repos for agent-readiness. Avatar, grade badge, ranked repo list.

Run agentkit user-scorecard github:username to generate a shareable profile card.
